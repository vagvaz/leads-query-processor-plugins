The New 2013/14 FFF Home Kit

ULTIMATIVER TRAGEKOMFORT BEI ALLEN BEDINGUNGEN


Das 2013/14 FFF Match Herren Fußballtrikot ist das gleiche Trikot, wie es die Spieler auf dem Platz tragen und bietet elegante Frankreichdetails, ein einzigartiges Denim-Design und ein umfassendes Kühlungssystem für Tragekomfort und Style. 


Vorteile

  • Schweißableitendes Dri-FIT-Material mit dem Tragegefühl weicher Baumwolle für kühlen Tragekomfort
  • Lasergeschnittene seitliche Einsätze für zusätzliche Belüftung an den entscheidenden Stellen
  • Spezielle Mesh-Einsätze auf der Rückseite für zusätzliche Atmungsaktivität
  • Verschweißte Knopfblende für ein modernen Look und ein tolles Tragegefühl
  • Authentisches Design mit gesticktem Wappen und Teamdetails

Produktinformationen

  • Hergestellt aus recycelten Plastikflaschen
  • Material: Einsatz vorne: Dri-FIT 96 % Polyester/4 % Baumwolle aus biologischem Anbau. Rückeneinsatz/Kragen: 100 % Polyester. Innenmaterial vorne: 100 % recycelte Baumwolle.
